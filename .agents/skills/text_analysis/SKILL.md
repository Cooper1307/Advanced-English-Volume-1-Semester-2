# 课文深度分析技能

## 触发方式

当用户请求分析某篇课文时使用此技能。常见触发语：

- "分析这篇课文"
- "精读 XX 页"
- "帮我做课文分析"

## 分析流程

### 第 1 步：通读全文，提取元信息

- 确认课文标题、作者、出处
- 概括文章主旨（1-2 句话）
- 标注文体类型（议论文 / 记叙文 / 说明文 / 散文等）

### 第 2 步：篇章结构分析

- 划分段落层次，归纳各段大意
- 梳理行文逻辑：时间顺序 / 空间顺序 / 逻辑推进 / 对比等
- 找出主题句（topic sentence）及过渡句

### 第 3 步：词汇与短语

- 提取重点词汇（建议 15-30 个），附词性、释义、例句
- 标注固定搭配和习语
- 归类同义词 / 反义词

### 第 4 步：句法分析

- 选取 3-5 个长难句进行拆解
- 标注语法要点（从句类型、非谓语、倒装、虚拟语气等）
- 给出逐层分析和中文翻译

### 第 5 步：修辞手法

- 识别并举例说明文中使用的修辞手法
- 常见关注点：比喻（metaphor / simile）、讽刺（irony / satire）、排比（parallelism）、象征（symbolism）、夸张（hyperbole）、委婉（euphemism）

### 第 6 步：翻译难点

- 标注文化负载词（culture-loaded words）及翻译策略
- 分析双关语、隐喻等不可直译的表达
- 给出参考译文及翻译技巧说明

### 第 7 步：思考与讨论

- 基于教材 Guiding Questions 展开深度解读
- 补充延伸讨论话题
